# Contributing to libvips_ffi

Thank you for your interest in contributing to libvips_ffi!

## Current Status

⚠️ **This project is currently in early development stage.**

At this time, we are **not accepting pull requests** from external contributors. The API and architecture are still evolving rapidly, and we need to stabilize the core functionality before opening up to community contributions.

## What You Can Do

While we're not accepting code contributions yet, you can still help in the following ways:

- **Report bugs**: If you find a bug, please open an issue with detailed reproduction steps
- **Request features**: Share your ideas and use cases by opening a feature request issue
- **Ask questions**: Use GitHub Discussions for questions about usage or implementation
- **Star the project**: Show your support by starring the repository

## Future Contributions

Once the project reaches a stable state, we plan to:

1. Open up for community contributions
2. Publish contribution guidelines
3. Set up a proper code review process

Please watch the repository for updates on when we start accepting contributions.

## Contact

For any questions, please reach out through GitHub Issues or Discussions.
